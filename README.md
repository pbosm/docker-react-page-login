# Configurando ambiente

- Git clone nesse repositorio;
- Navague até o dir
- Suba seu Docker
- Dê um docker-compose up
- Abra o navegador em localhost:3000

## Sem docker

- Git clone nesse repositorio;
- Navague até o dir
- npm i
- npm start
- Abra o navegador em localhost:3000
